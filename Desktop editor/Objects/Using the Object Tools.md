# Using the Object Tools

With the Worlds desktop editor, you can edit objects in your scene easily and intuitively, with easily accessible buttons on a single toolbar to toggle between different tool functions.

![Object tools in the Desktop Editor](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/506484005_739727315231889_435453471259630208_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=SCqrA99hNYQQ7kNvwHBNdVT&_nc_oc=Adl5kDv7-rLuj1Xa3hSdQgEoCQENbCtW0WY6Ex1Ou9GQn_35mKu0hBYqxwe8k2N5gvM&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfniwoD-QX8O0uwRbiPt4fBDPmmT5IZKXEsiouz0nE9Fww&oe=694BCA5E)

## Select object

![The Select tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/483681928_672113471993274_2926194712437271083_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=rwG2kxwiKAUQ7kNvwHWamSI&_nc_oc=AdnhAyex_HeWZYdKLe4H1zcQJmvdXE7vOspmBfbgK9shKfU_I0rpVmJEYBlUbK6cdSI&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_Afncqp5c49vwTTVXTa4b1UUE5JaXxVIKCxPEppeXE3UG-A&oe=694BE797)

Keyboard shortcut: **Shift** + **Q**

Select an object by selecting it in the **Hierarchy** panel, or selecting it in the **Scene** view. If another tool has been selected, click the **Select** tool while focus is on the object. You can also select more than one object at a time by holding down the **Shift** or **Ctrl** keys and selecting the objects in either the **Scene** view or the **Hiearchy** panel.

![Selecting an object](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452816722_512524734618816_171440519835222012_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=xDK1REezmskQ7kNvwHHCh9n&_nc_oc=AdnLi_7Ii1DdOXjNxkDJmClKBrUFCDaLXAWbkMWhJoQJ_uucsq2LcEnrGJEpqu4tCOE&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfkVMw1EzHK9yqbst8FSq4dyDbNJRlSPkyJigPACwI8iog&oe=694BDBD0)

## Move object

![The Move tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/506223332_739727318565222_6106341238701362507_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=y9bnFOH0DbIQ7kNvwEGYlno&_nc_oc=AdlNLOdV3vKJOcfs0FT_2YZVRBIybmFxAEXmshLiOhCTnIlHU_5Y_aRokXb5sR6A-9M&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfmlqW8TpmrKIsyhDIeD-hYqri761Fz3jD4hgMupVZAr9w&oe=694BD92E)

Keyboard shortcut: **Shift** + **W**

When you click the **Move** button with an object selected, a small three-dimensional (3D) coordinate system appears on the object.

![The 3D object coordinate system](https://scontent-dfw5-3.xx.fbcdn.net/v/t39.2365-6/506351661_739727308565223_3601343526414039856_n.png?_nc_cat=109&ccb=1-7&_nc_sid=e280be&_nc_ohc=1D9BRA5vlKIQ7kNvwFjNc3y&_nc_oc=AdkUxyjSuhfLzDmf9HDIWw02vn4e9tB2miN8GGbCrfqK804KDEZtzPJAYVUhHYdvn8M&_nc_zt=14&_nc_ht=scontent-dfw5-3.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfmUwtohRM5scpUXXX4DP8Z7Y3SLFbi2YMgSAx1OCWR-CA&oe=694BCC64)

It has arrows going along the red X (left-right), green Y (up-down), and blue Z (forward-back) directions. You can move the object in any of those directions by clicking and dragging on one of the arrows.You can move the object along the XY, XZ, or YZ planes by dragging one of the planar handles (the squares where the arrows meet in the center).

![Moving an object](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/453001218_512524691285487_6705275935354642462_n.png?_nc_cat=104&ccb=1-7&_nc_sid=e280be&_nc_ohc=gpH_p9wEwlEQ7kNvwF_2zwD&_nc_oc=AdkrjYm28WzkIBwkZciNz9LzF5FjTq8mmpTRjLcuBMIdISSjP_uMYJeJ0FHOt2Xc8Hk&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_Afm6muoZPZQfamupyIVtsxsDRf3GchMb4UZAoI0GYe77mg&oe=694BC75A)

## Rotate object

![The Rotate tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/483831424_672113465326608_6267010467848950135_n.png?_nc_cat=101&ccb=1-7&_nc_sid=e280be&_nc_ohc=8buxPH7YKV0Q7kNvwHyMZ5N&_nc_oc=AdkmJ3n40Y6qPTxEgU5TzmVqgPv58a81mpbajBBpOx-RoktzH9tVKpSAXpUvaOnrzzM&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfnqAwDsOwRvwVu2zTlOCQVSM8Rb-wUosJAZ3vlZfSH6fA&oe=694BBE72)

Keyboard shortcut: **Shift** + **E**

When you click the **Rotate** button with an object selected, a small three-dimensional (3D) set of angles appears on the object.

![Rotational angles](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/506330229_739727311898556_6957262673809842414_n.png?_nc_cat=100&ccb=1-7&_nc_sid=e280be&_nc_ohc=XDwxSt55k-wQ7kNvwFgBEWX&_nc_oc=AdnmfKeJ3pDDs0v4KrqooxmkHXL9twJg10I2kszFFz-L2A48S2s4pEh2YDedWZgZ0p0&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AflB2W8rrTcTG8QxJbUEb65E9-DIhdXpZ3Xae-0gTKtEjA&oe=694BDF14)

Rotate objects around the X, Y, or Z axes by dragging the red (X), green (Y) or blue (Z) angles. These partial circles show the rotation around either the center or the pivot point of the object (whichever you’ve chosen with the **Pivot** tool.) You can rotate the object in any of those directions by clicking the angle and dragging it so that the object rotates the desired amount.

![Rotating an object](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452977393_512524697952153_1776562809217327607_n.png?_nc_cat=110&ccb=1-7&_nc_sid=e280be&_nc_ohc=3yQZYbk9oa8Q7kNvwGFETDl&_nc_oc=AdlkbcvTYGGef06UnsirOLPKZfKuBQM7Lr7tMUitDDfMA4YTe2d5hf5S2-tElCoQFs8&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfnEJiJzp8_HOFVq2XtbNn6WNP9gSRHulpzbKbSa6ButcQ&oe=694BE90B)

## Scale object

![The Scale tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/483767042_672113468659941_649638574019667045_n.png?_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=QNJwPkEGnUQQ7kNvwEciNnt&_nc_oc=AdlNxFwUCb5wVqrmDcq3vxS3dnI70ezGRtlrfcGIwgayQNwkc7DWFv4_O25lRMzkQaw&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AflEmVG1Nd0anmlHKZcaACsM3D3CKZv8dR4WW81apf4JAA&oe=694BD518)

Keyboard shortcut: **Shift** + **R**

Scale objects along the X, Y, or Z axes by dragging the red (X), green (Y) or blue (Z) arrows. Doing this will only change the scale along that single axis. If you want to uniformly scale the object, drag the center gray box, or press **Shift** and drag one of the red, green, or blue handles.

![Scaling an object](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452882502_512524704618819_4306826134347465696_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=TooEsC4Xz7AQ7kNvwHFj4eD&_nc_oc=Adll0V9MkGXpJHO1ccMwlIM5lOuHVCE7xH4Grg68-iAUzL6iRQ3TkI2koZ3hK8FNBa0&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfkwjMFzXQkHvp47vhlQUpFLlfg7JDZIAdsMBWXgmi8siQ&oe=694BD679)

## Global / local coordinates

![The Local and Global Coordinates tool](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/476017273_645227211348567_7028385613716749596_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=sJXKGwb8Y8AQ7kNvwHHv7bT&_nc_oc=AdmaRFutH28c_7QnIR1mIINo3SxAWaaXKF4PCahj5Qg4frvtY6zTEYa_dXChkfucRJo&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AflZJ9QF1X5Nt5bvwhcuQ_VMevLhA-YpHWd56ABRXNHuTA&oe=694BDAEA)

This option toggles between **Local** and **Global** axes for the **Rotate** and **Move** tools. If this is set to **Local** (the default), any movement or rotation along (or about) an axis will be relative to the current orientation of the object. If it’s set to **Global**, it will move or rotate relative to the world’s X, Y, and Z axes.

![](https://scontent-dfw5-3.oculuscdn.com/v/t64.5771-25/38982716_1322793558690369_7686788648962514108_n.png?_nc_cat=109&ccb=1-7&_nc_sid=e280be&_nc_ohc=UcTmTyWpOuMQ7kNvwElbbLV&_nc_oc=AdmE6dwiR_hstOS5BhJsKoePokPI4YDGHOJRFMZAhqeZbypUYgaXJRX0HrMnGdc10Ck&_nc_zt=3&_nc_ht=scontent-dfw5-3.oculuscdn.com&oh=00_AfmayXa09Y4H8WKkIXXi9o8pYa1rZbMaE70SSnFmxq6djg&oe=694BD187)

## Pivot

![The Pivot tool](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/475330529_641654821705806_7637903699027620163_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=0FS_x_4e4uAQ7kNvwEXkUIP&_nc_oc=Adld4dNztKZsD9OsSTbVCTPGNsrHuL60MalESkxxKApGjtUhtdCUthFLsofuqjneu3Y&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfmUqT6XPNQhUgxxS3IPeGsPrTn4zXw68pUwyYGzYDlHfg&oe=694BDBCE)

By default, objects are positioned based on their center point. You can use the **Pivot** tool to position objects based on their pivot point, which is usually located at the bottom of the object, rather than based on their center point.

## Snapping tools

![The suite of Snapping tools](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/475098719_641654771705811_5416035882102001496_n.png?_nc_cat=106&ccb=1-7&_nc_sid=e280be&_nc_ohc=0owVbxjrlfQQ7kNvwF6HJF7&_nc_oc=AdnoQ6AR31grp9zYVoVwZIQVr5Bn1nHEC9vpA8wUt6aMCswStv1dRyLIqSKII8fTefQ&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfmaH_rfY3WZCTBOFs1e86HJiGLUn_mIntp2YZiIoO-1ig&oe=694BDB58)

You can use the **Snapping** tools to precisely translate, rotate, and scale objects in your scene by forcing the object to *snap* to a whole value, into a new position, orientation, or scale.

The following snapping tools are availabe in the Desktop Editor UI:

* [Translation Grid Snap](Using%20the%20Object%20Tools.md#translation-grid-snap)
* [Rotation Angle Snap](Using%20the%20Object%20Tools.md#rotation-angle-snap)
* [Scale Snap](Using%20the%20Object%20Tools.md#scale-snap)
* [Relative or Absolute Snap](Using%20the%20Object%20Tools.md#relative-or-absolute-snap)
* [Snap to Surfaces](Using%20the%20Object%20Tools.md#snap-to-surfaces)

### Translation Grid Snap

![The Translation Grid Snap tool](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/475055012_641654865039135_5589460192573784208_n.png?_nc_cat=102&ccb=1-7&_nc_sid=e280be&_nc_ohc=Cr3DnmFdoRwQ7kNvwFHMOar&_nc_oc=AdmuHfWFVWv1Y4ta-YkVrvHSci-uLnSY3lXSg4guLRQPWqLsebeoFf47e4OWSPFn36I&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfkiqvkAVwkFuM3Srw7in0r2zKaYfwLAfT7j7Ls3fNqmWA&oe=694BBB18)

**Translation Grid Snap** provides you a way to move objects along a given axis by specific increments called *Grid Snap Units*. They then snap into position along a whole coordinate value. Snapping objects into position helps you place object precisesly in your scene.

The Translation Grid Snap supports both local and global coordinate systems.

### Rotation Angle Snap

![The Rotation Angle Snap tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/475051369_641654845039137_4541661524071071907_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=utmCcV6M9BgQ7kNvwFD6jMn&_nc_oc=AdnOiMqGZMpU023dphsoYpC6DF4WfAr3DIWpsL_7ydScwzBVNRiALIqh_B2Cueeb360&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AflzHrWjEGV-SicDU-At4vBGFm0ZPBwnV6wP7YE3s3Vwqw&oe=694BE6C7)

Use the **Rotation Angle Snap** tool to rotate an object about its center point in specific increments.

### Scale Snap

![The Scale Snap tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/475766125_641654828372472_6757575579395757826_n.png?_nc_cat=110&ccb=1-7&_nc_sid=e280be&_nc_ohc=KRckLtSF-K4Q7kNvwFmpbp2&_nc_oc=Adm4ZnVkwgCHnLH7C7EXQgyxVuvKkUTMOh0tdf99L1ZBWGncQlyZsS687ES6D0Xtr0s&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfmdsaWSzl6UKIGHIsVdjXQguqWJMc61WP58bQVpri27Jg&oe=694BE630)

Use the **Scale Snap** toggle when you want to change an object’s size, and you want to scale it incrementally using a scaling factor.

### Relative or Absolute Snap

![The Relative or Absolute Snap tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/475096644_641654831705805_4094001368860514837_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=SLkeBOUItykQ7kNvwHea1_D&_nc_oc=AdkV0DJl6Wen85He1wsSqxnITjeZ7Zz-3h9X0k1AktsIgwMsYWz6CxjERimHrl7F_do&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AflXcnCO3OhZKxpMlYN41ShL33rA8IO4hAsMwMa4VSdVgg&oe=694BB817)

Snapping objects to values helps you align them precisely. By default, the desktop editor uses Relative snapping, but you can change it to use Absolute snapping.

| Snap Type | Description |
| --- | --- |
| Relative | Aligns objects based on their current position relative to other objects. This is useful to maintain consistent spacing or alignment between multiple objects in your scene. |
| Absolute | Aligns objects to a fixed grid or axis, regardless of their current position. This is useful to place objects at exact coordinates, or to align them to a specific grid. |

![The Snap to surfaces tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/474919481_641654825039139_6959945064584491108_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=Th_n1BKqTTgQ7kNvwFWhPca&_nc_oc=Adk1XEpM_HR8EEaHbYWZvBfhtbHRJa52tDN8dn8rgHRP34tDTnqdQb1_ny6gpvfxqiA&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=lzM9bx5kzCKpYB74pk3UOw&oh=00_AfmYH3vHczaSmRWMlKnWoHfJj0YVrva5S-_B9fEB4ATUdQ&oe=694BC059)

Use the **Snap to surfaces** to snap the pivot point of one object to the collider (surface) of another object in the scene.
