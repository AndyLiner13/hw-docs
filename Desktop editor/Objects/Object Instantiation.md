# Object Instantiation

- Click on the “Build” box icon from the top bar.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452521393_512524727952150_5511298057309237082_n.png?_nc_cat=104&ccb=1-7&_nc_sid=e280be&_nc_ohc=M29YxGENQaYQ7kNvwE9uhxQ&_nc_oc=AdmkEQ5ivBd4UitBq4j3l2QV6VrwL0q_txJcbVIvomjS9pOxYF061dXLMKj76B-2VjU&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=1ED6y3R45cWAnzkK8hnxAA&oh=00_AfiAbRI3A5Bi7rGf_wdbMihVXuaAca4ACx9dArRej1jJYA&oe=693117D2)
- Selecting shapes, gizmos, sounds from the drop-down menu will display 3D objects.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452934094_512524687952154_2607758860232986687_n.png?_nc_cat=102&ccb=1-7&_nc_sid=e280be&_nc_ohc=RcrkbVWDKV0Q7kNvwFJHCaM&_nc_oc=AdlMkoLyFJqXwbx0jf2cMTSK05g3csX_MW_oUEDfhvHLHh-B8NxU7IZVPX3vq_O-pi8&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=1ED6y3R45cWAnzkK8hnxAA&oh=00_Afj2uDQrn7K2yyMcWlZW4Xw3RGGmbvdMX2DZB5Lc7lAhVA&oe=693114F1)
- When the user clicks on a 3D object, the object is spawned in the world.
