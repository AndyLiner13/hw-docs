# Performance Metrics for Custom UI

There are 4 real-time metrics available to help with debugging Custom UI.

They are:

- Total # of visible UI
- Total # of value bindings
- Total data size of bindings (in KB)
- Binding update # in every frame

All the metrics can be seen under the [realtime metrics](/hw-docs/Performance/Performance%20tools/Enabling%20and%20modifying%20the%20real-time%20metrics%20panel%20in%20VR.md) list.
