# Editing Quests/Leaderboard/Variable Groups

You can use the Desktop Editor to view, create, edit, delete, debug, sort, and search quests, leaderboards, and variable groups, just like you can in VR.

## Getting Started

The first step for using all procedures in this article is to open the **systems panel**:

- Click the **Systems** dropdown in the top bar.
- Select the feature you want to modify.

## How to Edit

> **Note:** The following instructions focus on quests, but you can use the same steps for leaderboards and variable groups.

- Hover over the quest that you want to edit.
- Click the edit icon.
  ![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452653143_512511034620186_5486663686874101702_n.png?_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=HatZeXscclAQ7kNvwHbgvPa&_nc_oc=Adn-St46HsfldFwPZz90xcqwXUsVxYoDebH7kxo8KDaJAQur9-WxAL7PVKB8jObyWp8&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=zOYelhIYyqnxdTuZl1uXNg&oh=00_AfgYdKhz69yFuZe6kKdjbFF3ICmRAuZLz7R9d2eYx-iUPA&oe=692EAC7F)
- For variable groups, you will need to hover over the variable group, click on the overflow menu, and then select “Edit variable group”.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452587720_512511037953519_5965662815128967740_n.png?_nc_cat=100&ccb=1-7&_nc_sid=e280be&_nc_ohc=XWcx6ZBJeyoQ7kNvwG99teB&_nc_oc=AdmVoMOHx8vsDpViWQT_LyztN3eZ-VXpzprS4BZ4TOOIpO79-a8MhAZhTyrY1e7dQQI&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=zOYelhIYyqnxdTuZl1uXNg&oh=00_AfhF2-Oyzxvt4SQ7KxfQAmp5EF57dpGyonydYADCcWEF_w&oe=692EA882)
- Make any changes you want.
- Click ‘Save’.
