# Debugging Quests/Leaderboards/Variable Groups

You can use the Desktop Editor to view, create, edit, delete, debug, sort, and search quests, leaderboards, and variable groups, just like you can in VR.

## Getting Started

The first step for using all procedures in this article is to open the **systems panel**:

- Click the **Systems** dropdown in the top bar.
- Select the feature you want to modify.

## How to Debug

> **Note:** Unlike most features in the Systems panel, debugging is available only for quests and variable groups. The following instructions focus on quests, but you can use the same steps for variable groups.

- Click the “Debug Quests” icon.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452519016_512500317954591_542877101680441166_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=gD0R6p2fj_8Q7kNvwGsOHJp&_nc_oc=AdnObF0I2qu63Yba_lmZRAbuCpyVwW-uQCDyADPSqoMU-bo0QBTR8CQVr_bauibpz-g&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=c-UJhIdQlWnOAKfJBUKasw&oh=00_AfhDjvZ36iMJZWXHwqlxHl_OlHW0XWRJbFXFLKl0iiE7_w&oe=692EB3C4)
- Make any modifications required.
- You can also reset all values back to the default value by clicking “Reset all quests”.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452818117_512500187954604_3777672945376807269_n.png?_nc_cat=104&ccb=1-7&_nc_sid=e280be&_nc_ohc=javsWslnGnQQ7kNvwFSo65H&_nc_oc=Adnwtd-khpQp_7FjyqDPf0IvfyEPWdV-rjN3UJPVuyRmgmbgXPQspcDcAXS5vZ6H-JQ&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=c-UJhIdQlWnOAKfJBUKasw&oh=00_Afh9jcp7_FJrGPu9uVN-oGw-6CNqHZcyMr6toNwu4BDohQ&oe=692E9C16)
- Click ‘Save’.
