# Managing Persistent Variables Associated with a Variable Group

You can use the Desktop Editor to view, create, edit, delete, debug, sort, and search quests, leaderboards, and variable groups, just like you can in VR.

## Getting Started

The first step for using all procedures in this article is to open the **systems panel**:

- Click the **Systems** dropdown in the top bar.
- Select the feature you want to modify.

## Manage Persistent Variables

- Open the variable groups panel.
- Click on the variable group with the persistent variables you want to view.
  ![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452932800_512510237953599_2014147038568467266_n.png?_nc_cat=110&ccb=1-7&_nc_sid=e280be&_nc_ohc=P_Jl5LxpMMYQ7kNvwEkhyWI&_nc_oc=Adly1efLZj78wVgK7LTcLtni13UE6tSXGHBohP2EnZx3Ap-QYXXVBIq32c4UfHCRkcA&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=Dqb8IJ7YEupx9xIKsCnj5w&oh=00_Afj4jVZdI-DGh9hIWOAm3-dUaX9AO9wxZmIemiMLURwY9w&oe=692EADE0)
- You will now see the persistent variables associated with this group.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452935712_512510234620266_1791303906612712476_n.png?_nc_cat=106&ccb=1-7&_nc_sid=e280be&_nc_ohc=tpSzIgRRUGwQ7kNvwEx-T4f&_nc_oc=AdkopM1QhgSRUBcQcyNCrisWoQKERbtBcrr2d1aSbGMPAW8vspjfYozhwjLL4KXF1WY&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=Dqb8IJ7YEupx9xIKsCnj5w&oh=00_Afj8CeU9UuDGu2dmmVbh3hw5kMiXQ12w12w4i5UeVdrhpQ&oe=692EAF7A)
