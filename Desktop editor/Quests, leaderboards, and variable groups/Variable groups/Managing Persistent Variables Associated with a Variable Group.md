# Managing Persistent Variables Associated with a Variable Group

You can use the Desktop Editor to view, create, edit, delete, debug, sort, and search quests, leaderboards, and variable groups, just like you can in VR.

## Getting Started

The first step for using all procedures in this article is to open the **systems panel**:

- Click the **Systems** dropdown in the top bar.
- Select the feature you want to modify.

## Manage Persistent Variables

- Open the variable groups panel.
- Click on the variable group with the persistent variables you want to view.
  ![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/586164162_869065615631391_1170761457055257986_n.png?_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=Y-8pB8MAQ88Q7kNvwF5v-Gj&_nc_oc=AdmuKP1pQXML-gvOFkWpQbnGPc7JBAvwclXjJfgqDFUSlIrr0-iOvvw0DjyL8xq1pjk&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=cBOuCs2pCKRsHI1KgtEDhA&oh=00_AflB7CvihdTetdTWp4uIdEcW3q7VvXK-axA9ezD8gofmzg&oe=694BD2A8)
- You will now see the persistent variables associated with this group.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/584297474_869065618964724_8066526269860395513_n.png?_nc_cat=104&ccb=1-7&_nc_sid=e280be&_nc_ohc=WO26pfrcgF8Q7kNvwFmcJ9j&_nc_oc=AdlcxDgf6oKudlyL3BO6NBp2PzQpACwAhXHZIvx20pO70PQQ-J2ltZkxFdZhDMbBO6E&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=cBOuCs2pCKRsHI1KgtEDhA&oh=00_AfkYqhn12XesT04Tkozaat5L5f_9_PRv5Bo3VYY-RWdO7g&oe=694BD44B)
