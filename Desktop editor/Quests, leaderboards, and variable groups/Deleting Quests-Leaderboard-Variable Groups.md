# Deleting Quests/Leaderboard/Variable Groups

You can use the Desktop Editor to view, create, edit, delete, debug, sort, and search quests, leaderboards, and variable groups, just like you can in VR.

## Getting Started

The first step for using all procedures in this article is to open the **systems panel**:

- Click the **Systems** dropdown in the top bar.
- Select the feature you want to modify.

## How to delete quests

- Hover over the quest that you want to delete.
- Click the delete icon.
  ![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/586933411_869065605631392_5062164463822506357_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=sKFLhh_KlsoQ7kNvwGGaoQw&_nc_oc=AdkYN88gVbC-RbR-OJtCAyWddDnE123zdJwFbz-LFyOt-5qGhyLAbPO3DGmpKzv_-4U&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=-FuosFVEK7ktJP7HJW4ERg&oh=00_Afmh7unqy3wU-En2ydkl6cOkjAoTIveakP6fv2fSH4D93Q&oe=694BD948)
- Click ‘Delete’ on the modal dialog that opens.

## How to delete leaderboards

- Hover over the leaderboard that you want to delete.
- Click the delete icon.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/582347747_863661422838477_6853025179146449286_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=aq0SXCJNNhcQ7kNvwEoyp5l&_nc_oc=Adn2XKZyk6t9iM8a4e3j2QlMKE78trSy6EYEr1_OZ7BOYDZ_li68YSFswZgWSMqVru4&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=-FuosFVEK7ktJP7HJW4ERg&oh=00_AfkPg-DOEahHaA4JDvLq08TLr8yzguDC5CnXZwUXwbHYGg&oe=694BD956)
- Click ‘Delete’ on the modal dialog that opens.

## How to delete variable groups

- Hover over the variable group that you want to delete.
  ![](https://scontent-dfw5-3.xx.fbcdn.net/v/t39.2365-6/581837032_863661429505143_4325328526368508183_n.png?_nc_cat=109&ccb=1-7&_nc_sid=e280be&_nc_ohc=K2-Sr4WfyxEQ7kNvwGCT0Mv&_nc_oc=AdnWt_yB4hQdWDxkDxP2ka8NRKMSA23D_BOu3THrUHnLpV7nbLuc3iJuU9893IUax6U&_nc_zt=14&_nc_ht=scontent-dfw5-3.xx&_nc_gid=-FuosFVEK7ktJP7HJW4ERg&oh=00_Afla304kqzT3XR6bQVhYwdEor_UMR6ovftFbQ4ruR3MZ0w&oe=694BCF62)
- For variable groups you will need to hover over the variable group, click on the overflow menu, and then select “Delete”.
- Click ‘Delete’ on the modal dialog that opens.
