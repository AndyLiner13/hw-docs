# Deleting Quests/Leaderboard/Variable Groups

You can use the Desktop Editor to view, create, edit, delete, debug, sort, and search quests, leaderboards, and variable groups, just like you can in VR.

## Getting Started

The first step for using all procedures in this article is to open the **systems panel**:

- Click the **Systems** dropdown in the top bar.
- Select the feature you want to modify.

## How to delete quests

- Hover over the quest that you want to delete.
- Click the delete icon.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/581800178_863661419505144_1572659553470463145_n.png?_nc_cat=106&ccb=1-7&_nc_sid=e280be&_nc_ohc=imfzTD-tN8EQ7kNvwFrpm5I&_nc_oc=AdlfNNezxsfX66A0BdWyd8n_wz_qRixUOjU8V3CwZCcdxIiV3vw1JjzMS1ShHV5VAbI&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=LWyhMAvKJ9p0Rb8jyWgDOQ&oh=00_AfhKHiZQSJ_npgUpA21FMXlO-fl50oC9uHyOxWNUVbHkzA&oe=693132BE)
- Click ‘Delete’ on the modal dialog that opens.

## How to delete leaderboards

- Hover over the leaderboard that you want to delete.
- Click the delete icon.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/582347747_863661422838477_6853025179146449286_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=1R-4vJ6kY74Q7kNvwELIlga&_nc_oc=AdlmRNWG6mBh4DqJK67eO7vP9-bz8u3vErpqwpU5QWTBLtTZKjQZAvodavO0PXVJizo&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=LWyhMAvKJ9p0Rb8jyWgDOQ&oh=00_Afh6lBzLNkl3OJlw8z2TszrTqiIS0CYG8UMMxit8xHPfOw&oe=69314316)
- Click ‘Delete’ on the modal dialog that opens.

## How to delete variable groups

- Hover over the variable group that you want to delete.
  ![](https://scontent-dfw5-3.xx.fbcdn.net/v/t39.2365-6/581837032_863661429505143_4325328526368508183_n.png?_nc_cat=109&ccb=1-7&_nc_sid=e280be&_nc_ohc=RqfcX2Iwks8Q7kNvwEKW5aq&_nc_oc=Admf7g1ofCDkiaQV5NMlRueRhClr4m2Wk4BEmoljMSAOqiXXI0WnZrSZF4vCcJCxMmU&_nc_zt=14&_nc_ht=scontent-dfw5-3.xx&_nc_gid=LWyhMAvKJ9p0Rb8jyWgDOQ&oh=00_AfhGDi1xeHWr2kpi_robHPN2gMvbhLIzSIZSlRjqIMGfCA&oe=69313922)
- For variable groups you will need to hover over the variable group, click on the overflow menu, and then select “Delete”.
- Click ‘Delete’ on the modal dialog that opens.
