# Create a New World

Opening the Meta Horizon Worlds desktop editor takes you to the editor’s **Creation Home**.

![The Creation Home page](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/480792958_656120606925894_833399565384578386_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=BAUIopRHsqMQ7kNvwGnDXbu&_nc_oc=AdmM_xpYzBuysToZOiS10U_cR5sUIc9Tmvowy8SQXqc9Atk5vCjDYbX6fZeYPURsoipMQ_EhC8eM7_OcVAYNg4CA&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=UQwjupVrBkrEaId9tq4K1w&oh=00_Afl9hGEgIJBtuz-CSzlbCGE4kbj2dRyclkTcxegwOZsnig&oe=6955994E)

From this page, you can open an existing world that you’ve created, or start creating a new one.

**To create a new world**

- Open the Worlds desktop editor.

  ![The Start button on the app icon](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/546833532_811560994715187_4355045804765376973_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=33q7c5yUa_UQ7kNvwGN_9lC&_nc_oc=AdknhElyqbvfof9_LZ8p3_v1r-HodQURB8e3ClTviZ_V4qpLVB_plCCq8qb3NaR1vb8yDzSNDZC6ZiwwCKzisjzP&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=UQwjupVrBkrEaId9tq4K1w&oh=00_AfkfGcn40SUQlv7jrbVJpvnWd2Vmh7XKOJzPAvYR1ietvA&oe=69558839)
- On the **Creation Home** page, click **New World**.

  ![The New World button](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/480692096_656120603592561_7321082811270461964_n.png?_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=MQwVClbcUXsQ7kNvwGx7FbO&_nc_oc=AdnJAa6UQC5g245QcgN8KHsj1JewLaO3BAqnASqUeBYSCE-cwyiGAlTXt20DK2oueWNWE-cFOhu6Wtz1lgB5mQv1&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=UQwjupVrBkrEaId9tq4K1w&oh=00_AflY65zld45RIutBC1HQ507cXY970UQwz8M_rKKT_trgWg&oe=6955971F)
- When the desktop editor opens, click **Rename World** from the main menu.

  ![The Rename World option](https://scontent-dfw5-3.xx.fbcdn.net/v/t39.2365-6/585041273_869065665631386_2506310454970661596_n.png?_nc_cat=108&ccb=1-7&_nc_sid=e280be&_nc_ohc=9SXE7NZSLPcQ7kNvwHWsgxN&_nc_oc=Adk8oFrC2clxP7N7FreI1tYnxsn-mHIJeY3UhLY_pOVuKAreWm_c0bkqHYSw3cdNdx2RHrgU-Ygm1F4xH1rDVZes&_nc_zt=14&_nc_ht=scontent-dfw5-3.xx&_nc_gid=UQwjupVrBkrEaId9tq4K1w&oh=00_AfkoDV7aaS9WVCFDyUbdkvL1ZWSEgrCflf6hZ5zaPeg-bA&oe=69558C8D)
- Provide a name for your new world

  ![Providing a name for your new world](https://scontent-dfw5-3.xx.fbcdn.net/v/t39.2365-6/480688964_656120580259230_806255238760088754_n.png?_nc_cat=108&ccb=1-7&_nc_sid=e280be&_nc_ohc=3KrARKrDjAEQ7kNvwHos-C3&_nc_oc=AdnJO3NXaGZu2zdQpCS65sPhei50KgCVE19SJeJaGe1YwZctU_6xfH2C8zy2ps3H0xE_15hR8pit5zivXFPqgJ_2&_nc_zt=14&_nc_ht=scontent-dfw5-3.xx&_nc_gid=UQwjupVrBkrEaId9tq4K1w&oh=00_AfnVJU6q6-xO4sB6m-Bw0gxwNpHhHzYeZM3Y_58M6YwtyA&oe=69558846)

  **Note**: If you do not provide a name for your new world, it will still exist and be saved as normal. However, the next time you open it, it will be called “New World” with the count of how many worlds you’ve created without names appended to it (for instance, “New World 4.” If you do this several times, you may end up with multiple worlds called by similar names, potentially causing confusion. You can always rename it to something else at a later time.
- Click **Save**.
