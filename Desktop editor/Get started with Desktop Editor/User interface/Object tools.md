# Object tools

![Object tools in the Desktop Editor](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/506484005_739727315231889_435453471259630208_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=wFz6Ik4GdzkQ7kNvwHbkOuq&_nc_oc=AdkdS3R0_eauiqTtgIOHTQxMZjigImENBBTGaenz6ZZuKd2rFp2dM7A2GkiLg57nsww&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_Afh8wVYCoZiLoK7z8r-pcgHwWCdEVD6uHHlvz_gOSovYnQ&oe=6931341E)

The object tools in the Desktop Editor provide a set of commonly used tools for building scenes and asset use. Each option provides a menu of different types of tools that you can use for creating your worlds.

For more in-depth information about the object tools, see [Using the Object tools](/hw-docs/Desktop%20editor/Objects/Using%20the%20Object%20Tools.md)

This suite of tools contains the following object tool tyou can use:

* [Select](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#select)
* [Move](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#move)
* [Rotate](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#rotate)
* [Scale](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#scale)
* [Local and Global Coordinate Systems](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#local-and-global-scripts)
* [Pivot](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#pivot)
* [Snapping tools](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#snapping-tools)

## Select

![The Select tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/483681928_672113471993274_2926194712437271083_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=bHft4cttjrIQ7kNvwEWVzbE&_nc_oc=AdnHVAU_pB-RFKoOhvmth6nZM_4Ueu0b0V-TaXT3zwN4i7vRSZC_WU-78ap9jkIFEhI&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_AfhOb4BQFfIEb0WpycgWcslM5wlFx1OYK4goI2yW0cT9cw&oe=69311917)

Use the **Select** tool to select an object on the scene. This enables you to easily manipulate the object or view its properties. You can also select multiple objects by either using the **Select** tool and holding the **Ctrl** key down while you select the objects in the scene, or by holding down the **Ctrl** or **Shift** key and selecting the objects in the **Hiearchy** panel.

## Move

![The Move tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/506223332_739727318565222_6106341238701362507_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=fQ8aZ5FqkjMQ7kNvwGZyCf-&_nc_oc=AdlSM2hW55P4gRIppSBT6H0PSq7aYMFyr2VII95e-CmfM2_RcX_y6XtnBVv58JReNy8&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_Afh_WkB0_NE4XgZUdG0-RHYGv0nxk3urWFsy2fzejADnTA&oe=693142EE)

Select an object and then use the **Move** tool to reposition it anywhere in your scene.

## Rotate

![The Rotate tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/483831424_672113465326608_6267010467848950135_n.png?_nc_cat=101&ccb=1-7&_nc_sid=e280be&_nc_ohc=k9cCOSFz4yMQ7kNvwH0B5dD&_nc_oc=Adl6FeG9hOFjDsMd9irHIvVLcN7u_znxlxEYlPkqEfAQjN1uOC_jGuGRWyC5fKjCjZY&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_AfiIIYsYi7cT5m-IFxgkLkqpgyMUtbNgYdbydflWaFBVDA&oe=69312832)

Use the **Rotate** tool to rotate an object about its center or pivot point. You can rotate it in any direction using the tool.

## Scale

![The Scale tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/483767042_672113468659941_649638574019667045_n.png?_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=P8T-e-FNvTkQ7kNvwF10kbx&_nc_oc=Adn0CZorwcB9zqaTEamy2vzU4RX05Gy1mp5JmnzLYrvLBvgzLKsXoFCHCwEQVHOY_2Q&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_AfiRYSIDIJZMFsWVgKLeQo9aVn-TlOfwNeAAIgd8VjLSMA&oe=69313ED8)

Use the **Scale** tool resize an object in your scene. It can be scaled in a single direction (such as making it longer or wider), or in all directions (making it bigger or smaller).

## Local and Global Coordinates

![The Local and Global Coordinate tool](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/476017273_645227211348567_7028385613716749596_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=Dgp-CttTqtUQ7kNvwH9q3Gr&_nc_oc=Adkc8862Zwmsklg6iuEJxlSWv6c7sF8DPYAMmUIABDV1txmP1o24xZ4M2mwVWIdUC8k&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_Afhfa56HEIE0pRf__O_FtzxtPn99ydlNnGqWZWNUxJK5QQ&oe=693144AA)

This option toggles between **Local** and **Global** axes for the **Rotate** and **Move** tools. If this is set to **Local** (the default), any movement or rotation along (or about) an axis will be relative to the current orientation of the object. If it’s set to **Global**, it will move or rotate relative to the world’s X, Y, and Z axes.

## Pivot

![The Pivot tool](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/475330529_641654821705806_7637903699027620163_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=zJtsESgXnR4Q7kNvwG6KiWB&_nc_oc=AdnZfvz1PkoyxDzf1vYMjuVNwghwGvsFZMfksBeF9iwR8upFXcMViEu3S_yjRApoqKg&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_AfgVYsptgp919fpHwTryReQcFUmbT6eViyo88ubl-BrEsA&oe=6931458E)

By default, objects are positioned based on their center point. You can use the **Pivot** tool to position objects instead based on their pivot point, which is usually located at the bottom of the object.

## Snapping tools

![The suite of Snapping tools](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/475098719_641654771705811_5416035882102001496_n.png?_nc_cat=106&ccb=1-7&_nc_sid=e280be&_nc_ohc=rx28QbuYy68Q7kNvwFoob6A&_nc_oc=AdmY3cxtBdwb8tl9yCuZoR8J0DziSti154ElVOb9yUvyraMugXGFSHt88KrmtwjQJjM&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_AfhVp5Kj7L3xAUPDygrYuAstAeHFUNA6PLMdSmWd3VwWxA&oe=69314518)

You can use the **Snapping** tools to precisely translate, rotate, and scale objects in your scene by forcing the object to *snap* to a whole value, into a new position, orientation, or scale.

The following snapping tools are availabe in the Desktop Editor UI:

* [Translation Grid Snap](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#translation-grid-snap)
* [Rotation Angle Snap](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#rotation-angle-snap)
* [Scale Snap](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#scale-snap)
* [Relative or Absolute Snap](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#relative-or-absolute-snap)
* [Snap to Surfaces](/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#snap-to-surfaces)

### Translation Grid Snap

![The Translation Grid Snap tool](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/475055012_641654865039135_5589460192573784208_n.png?_nc_cat=102&ccb=1-7&_nc_sid=e280be&_nc_ohc=5MbukCpOV0gQ7kNvwE6jJTg&_nc_oc=AdkNKtZC1FpFTFS1ASmjnZDVZ3HPo29PKEH8vNGnPRXPk4s7SpcQ-SZny3WkxETQ_iw&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_AfhQ3GjGfUdjm5A3bu5zaRU53z0oaz_EIiMO7Fb2IN5SXA&oe=693124D8)

**Translation Grid Snap** provides you a way to move objects along a given axis by specific increments called *Grid Snap Units*. They then snap into position along a whole coordinate value. Snapping objects into position helps you place object precisesly in your scene.

The Translation Grid Snap supports both local and global coordinate systems.

![The Rotation Angle Snap tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/475051369_641654845039137_4541661524071071907_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=7bkdgAzXLOgQ7kNvwGqdvtf&_nc_oc=AdkpXu135iuGvUUPY_-wdMBkm_37klrvDjeaqXrqHdFShE7jQ6aRunDZq_x-AUURbC0&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_AfiBdy2Y5LyjQ0qot6zn069iXmXVjPpN3LXu23WoKCTJfQ&oe=69311847)

Use the **Rotation Angle Snap** tool to rotate an object about its center point in specific increments.

### Scale Snap

![The Scale Snap tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/475766125_641654828372472_6757575579395757826_n.png?_nc_cat=110&ccb=1-7&_nc_sid=e280be&_nc_ohc=dhkeKd-aCx0Q7kNvwFmJQ6l&_nc_oc=Adm9oBeQS4qNsVhfleStkUUpgMYtk9tdqbLhoeKosxZeu1Usya1RpnWFxwuRC4WiDuw&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_AfgtTBTqGQ3Awi68NNZbPdpu9PMYTBvuPXvvbSV_gyJR9g&oe=693117B0)

Use the **Scale Snap** toggle when you want to change an object’s size, and you want to scale it incrementally using a scaling factor.

### Relative or Absolute Snap

![The Relative or Absolute Snap tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/475096644_641654831705805_4094001368860514837_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=n0dK1wSv6dsQ7kNvwFwIy_m&_nc_oc=Adn7XGSoe-83GAudfR2S909KrgnUUUNV9nxN-8BkwF6gpC7o0C6-BoRHovGSc7vNfNc&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_AfhXE5q4lFfaG53bhwD2rh0xgTeikYIkIiHNvfHedls3hw&oe=693121D7)

Snapping objects to values helps you align them precisely. By default, the Desktop Editor uses Relative snapping, but you can change it to use Absolute snapping.

| Snap Type | Description |
| --- | --- |
| Relative | Aligns objects based on their current position relative to other objects. This is useful to maintain consistent spacing or alignment between multiple objects in your scene. |
| Absolute | Aligns objects to a fixed grid or axis, regardless of their current position. This is useful to place objects at exact coordinates, or to align them to a specific grid. |

![The Snap to surfaces tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/474919481_641654825039139_6959945064584491108_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=vfaf-1Eep6oQ7kNvwG-kJ3L&_nc_oc=AdlTNxjLD-VO5GwoymMBxzCm9qDMEmCVoWZf4B6pPngw2KSdF0L65EKteFj-9fjkg8I&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=44x6bVNtAK_OStYYh5t-jA&oh=00_AfgYGkFUqwESCKz-9_M_YpGuej3MIy_XRV1iLllLcG1Gxw&oe=69312A19)
