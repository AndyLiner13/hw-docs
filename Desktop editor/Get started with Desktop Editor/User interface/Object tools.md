# Object tools

![Object tools in the Desktop Editor](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/506484005_739727315231889_435453471259630208_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=SCqrA99hNYQQ7kNvwHBNdVT&_nc_oc=Adl5kDv7-rLuj1Xa3hSdQgEoCQENbCtW0WY6Ex1Ou9GQn_35mKu0hBYqxwe8k2N5gvM&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_AfmkPIeN_eqbFSQvpN8lzmnE4JdxUpUJKUIRhv6lelDepQ&oe=694BCA5E)

The object tools in the Desktop Editor provide a set of commonly used tools for building scenes and asset use. Each option provides a menu of different types of tools that you can use for creating your worlds.

For more in-depth information about the object tools, see [Using the Object tools](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Objects/Using%20the%20Object%20Tools.md)

This suite of tools contains the following object tool tyou can use:

* [Select](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#select)
* [Move](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#move)
* [Rotate](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#rotate)
* [Scale](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#scale)
* [Local and Global Coordinate Systems](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#local-and-global-scripts)
* [Pivot](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#pivot)
* [Snapping tools](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#snapping-tools)

## Select

![The Select tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/483681928_672113471993274_2926194712437271083_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=rwG2kxwiKAUQ7kNvwHWamSI&_nc_oc=AdnhAyex_HeWZYdKLe4H1zcQJmvdXE7vOspmBfbgK9shKfU_I0rpVmJEYBlUbK6cdSI&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_AfnhZPYEn6i3ny3bK0kbbB8KsfPbhn-LM9o0xYKEwJOOgQ&oe=694BE797)

Use the **Select** tool to select an object on the scene. This enables you to easily manipulate the object or view its properties. You can also select multiple objects by either using the **Select** tool and holding the **Ctrl** key down while you select the objects in the scene, or by holding down the **Ctrl** or **Shift** key and selecting the objects in the **Hiearchy** panel.

## Move

![The Move tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/506223332_739727318565222_6106341238701362507_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=y9bnFOH0DbIQ7kNvwEGYlno&_nc_oc=AdlNLOdV3vKJOcfs0FT_2YZVRBIybmFxAEXmshLiOhCTnIlHU_5Y_aRokXb5sR6A-9M&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_AfnjKgqxkJL09jAWHaj7lruCGI2-hhbP7rdXK8osvPcw5w&oe=694BD92E)

Select an object and then use the **Move** tool to reposition it anywhere in your scene.

## Rotate

![The Rotate tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/483831424_672113465326608_6267010467848950135_n.png?_nc_cat=101&ccb=1-7&_nc_sid=e280be&_nc_ohc=8buxPH7YKV0Q7kNvwHyMZ5N&_nc_oc=AdkmJ3n40Y6qPTxEgU5TzmVqgPv58a81mpbajBBpOx-RoktzH9tVKpSAXpUvaOnrzzM&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_Aflsft-J0vrybZ-S01Q2WnCKu8c4ycaYnVKjTUYMseH2IQ&oe=694BBE72)

Use the **Rotate** tool to rotate an object about its center or pivot point. You can rotate it in any direction using the tool.

## Scale

![The Scale tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/483767042_672113468659941_649638574019667045_n.png?_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=QNJwPkEGnUQQ7kNvwEciNnt&_nc_oc=AdlNxFwUCb5wVqrmDcq3vxS3dnI70ezGRtlrfcGIwgayQNwkc7DWFv4_O25lRMzkQaw&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_Afn11pnrZqWeFImO8DEJbsFr4ibmD3wrF0IvEQopqQTxKg&oe=694BD518)

Use the **Scale** tool resize an object in your scene. It can be scaled in a single direction (such as making it longer or wider), or in all directions (making it bigger or smaller).

## Local and Global Coordinates

![The Local and Global Coordinate tool](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/476017273_645227211348567_7028385613716749596_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=sJXKGwb8Y8AQ7kNvwHHv7bT&_nc_oc=AdmaRFutH28c_7QnIR1mIINo3SxAWaaXKF4PCahj5Qg4frvtY6zTEYa_dXChkfucRJo&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_AfkM7LmmrGCP0Pnvl3vNjGI4CeFVE3rBseP4LlxhpJZmAg&oe=694BDAEA)

This option toggles between **Local** and **Global** axes for the **Rotate** and **Move** tools. If this is set to **Local** (the default), any movement or rotation along (or about) an axis will be relative to the current orientation of the object. If it’s set to **Global**, it will move or rotate relative to the world’s X, Y, and Z axes.

## Pivot

![The Pivot tool](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/475330529_641654821705806_7637903699027620163_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=0FS_x_4e4uAQ7kNvwEXkUIP&_nc_oc=Adld4dNztKZsD9OsSTbVCTPGNsrHuL60MalESkxxKApGjtUhtdCUthFLsofuqjneu3Y&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_AfkNZqlXwikOcflKd75Cz9HCsjUSB3T8dTJL9xJg-MhFDQ&oe=694BDBCE)

By default, objects are positioned based on their center point. You can use the **Pivot** tool to position objects instead based on their pivot point, which is usually located at the bottom of the object.

## Snapping tools

You can use the **Snapping** tools to precisely translate, rotate, and scale objects in your scene by forcing the object to *snap* to a whole value, into a new position, orientation, or scale.

The following snapping tools are availabe in the Desktop Editor UI:

* [Translation Grid Snap](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#translation-grid-snap)
* [Rotation Angle Snap](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#rotation-angle-snap)
* [Scale Snap](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#scale-snap)
* [Relative or Absolute Snap](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#relative-or-absolute-snap)
* [Snap to Surfaces](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Get%20started%20with%20Desktop%20Editor/User%20interface/Object%20tools.md#snap-to-surfaces)

### Translation Grid Snap

![The Translation Grid Snap tool](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/475055012_641654865039135_5589460192573784208_n.png?_nc_cat=102&ccb=1-7&_nc_sid=e280be&_nc_ohc=Cr3DnmFdoRwQ7kNvwFHMOar&_nc_oc=AdmuHfWFVWv1Y4ta-YkVrvHSci-uLnSY3lXSg4guLRQPWqLsebeoFf47e4OWSPFn36I&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_AfmI_oSwJWqMhhIsu2P3XBB-qR7DCVcXtteT11wGDSZxag&oe=694BBB18)

**Translation Grid Snap** provides you a way to move objects along a given axis by specific increments called *Grid Snap Units*. They then snap into position along a whole coordinate value. Snapping objects into position helps you place object precisesly in your scene.

The Translation Grid Snap supports both local and global coordinate systems.

![The Rotation Angle Snap tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/475051369_641654845039137_4541661524071071907_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=utmCcV6M9BgQ7kNvwFD6jMn&_nc_oc=AdnOiMqGZMpU023dphsoYpC6DF4WfAr3DIWpsL_7ydScwzBVNRiALIqh_B2Cueeb360&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_AfmvYmrGpNomO0gjqAL7Z4rC-MXP_ENttqG8V_owxaNyDw&oe=694BE6C7)

Use the **Rotation Angle Snap** tool to rotate an object about its center point in specific increments.

### Scale Snap

![The Scale Snap tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/475766125_641654828372472_6757575579395757826_n.png?_nc_cat=110&ccb=1-7&_nc_sid=e280be&_nc_ohc=KRckLtSF-K4Q7kNvwFmpbp2&_nc_oc=Adm4ZnVkwgCHnLH7C7EXQgyxVuvKkUTMOh0tdf99L1ZBWGncQlyZsS687ES6D0Xtr0s&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_AflM-kcv8B6t-b6tAUrnnRm-1xi3Ked6c0A0Nwp1h2XyjA&oe=694BE630)

Use the **Scale Snap** toggle when you want to change an object’s size, and you want to scale it incrementally using a scaling factor.

### Relative or Absolute Snap

![The Relative or Absolute Snap tool](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/475096644_641654831705805_4094001368860514837_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=SLkeBOUItykQ7kNvwHea1_D&_nc_oc=AdkV0DJl6Wen85He1wsSqxnITjeZ7Zz-3h9X0k1AktsIgwMsYWz6CxjERimHrl7F_do&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=9bX4pdMa4aEhmN7tBtqeLA&oh=00_AfnJmtAwGG-A5RYY3s4FU-dO4QgZYztwUo8DA5RSjVbYdA&oe=694BB817)

Snapping objects to values helps you align them precisely. By default, the Desktop Editor uses Relative snapping, but you can change it to use Absolute snapping.

| Snap Type | Description |
| --- | --- |
| Relative | Aligns objects based on their current position relative to other objects. This is useful to maintain consistent spacing or alignment between multiple objects in your scene. |
| Absolute | Aligns objects to a fixed grid or axis, regardless of their current position. This is useful to place objects at exact coordinates, or to align them to a specific grid. |
