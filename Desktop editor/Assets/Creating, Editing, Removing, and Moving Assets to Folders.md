# Creating, Editing, Removing, and Moving Assets to Folders

## Creating folders

To create a new folder, click the “Add New” menu button above the left navigation column and select “Folder”.

## Editing folders

- To rename a folder, right-click the folder in the left navigation column and select “Edit”.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452846049_512500371287919_1064496452354703381_n.png?_nc_cat=100&ccb=1-7&_nc_sid=e280be&_nc_ohc=y_CqdypfP60Q7kNvwEPMYsl&_nc_oc=AdlVk85GgSd4StB8uWmGto7D5NYbT01HrHX64VVYdcIdVKKz0ULcKcGqS6nf4nA_wJo&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=TaPZVZ6Fe7lUOtFHjY8bkQ&oh=00_Afln29PUf3mxYrvgmRsn90R9Qnzb6HV_YNVve7TpCGABEQ&oe=694BE879)
- Fill out the Edit Folder fields and then click “Update” to finalize your changes.

## Removing folders

To remove a folder from your library, right-click the folder in the left navigation column and select “Remove”.
![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452742636_512500367954586_2406993045507463854_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=kwCoM3evkQoQ7kNvwGeVRE2&_nc_oc=AdmX-T5Y2UCYlOLypAZm7zEvXeDBoiwVCK1gNvhUXpHwdttApBAeOZu4uYKSvyBRm98&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=TaPZVZ6Fe7lUOtFHjY8bkQ&oh=00_AfkPqY5G7YnjFTIOozA5aeTUdUcjSqzvEoxm7Digqzj_Pw&oe=694BB462)

> **Note:** When deleting a folder, any assets inside will be moved into the “Removed Assets” folder.

## Moving assets to different folders

To move an asset into a different folder, right-click the asset card  and select “Move” and then fill out the modal dialog.
