# Creating, Editing, Removing, and Moving Assets to Folders

## Creating folders

To create a new folder, click the “Add New” menu button above the left navigation column and select “Folder”.

## Editing folders

- To rename a folder, right-click the folder in the left navigation column and select “Edit”.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452846049_512500371287919_1064496452354703381_n.png?_nc_cat=100&ccb=1-7&_nc_sid=e280be&_nc_ohc=xstQq3LrpXQQ7kNvwFYH1F-&_nc_oc=AdmiBMRynUqSXObAUugLYBIFCPbQBkh5bfnRyiYGCGmc58eYhYfApZHuJIbH5d7xT_A&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=wrQSaaB8F8lxv3_WM8eOMQ&oh=00_AfjIzHxzcRf6anDcD1zODB3sg6cMt2FGVIFQ368xLOYuTA&oe=692FC879)
- Fill out the Edit Folder fields and then click “Update” to finalize your changes.

## Removing folders

To remove a folder from your library, right-click the folder in the left navigation column and select “Remove”.
![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452742636_512500367954586_2406993045507463854_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=u5G587oRfoIQ7kNvwEOqFsv&_nc_oc=AdnkIo0VpwiR0SeMuWLTApQm67sXp0qZk-y4abziQ6Y0CHximpp1aciqtCh7WSejuuA&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=wrQSaaB8F8lxv3_WM8eOMQ&oh=00_Afjdv0L6s38-DZlhzTPcx5ocq-zQdXxdlPeafNhIYYmD6w&oe=692FCCA2)

> **Note:** When deleting a folder, any assets inside will be moved into the “Removed Assets” folder.

## Moving assets to different folders

To move an asset into a different folder, right-click the asset card  and select “Move” and then fill out the modal dialog.
