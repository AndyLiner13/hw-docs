# Creating, Editing, Removing, and Moving Assets to Folders

## Creating folders

To create a new folder, click the “Add New” menu button above the left navigation column and select “Folder”.

## Editing folders

- To rename a folder, right-click the folder in the left navigation column and select “Edit”.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452846049_512500371287919_1064496452354703381_n.png?_nc_cat=100&ccb=1-7&_nc_sid=e280be&_nc_ohc=TSi47N--m2QQ7kNvwE8Sq84&_nc_oc=AdkMPoqZXHU-NsoIbH4S_8_ilPIQxKDQVHvZDPuqkGNlu_L4852UZ3BEaKpVZ3zInuU&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=ry7IOLvf20Hf2Jru2SXokA&oh=00_Afj4QN7HUWlNOzLCXtJ0AZ2BypY8DZjeX7vJK0tqq3mY3A&oe=693119F9)
- Fill out the Edit Folder fields and then click “Update” to finalize your changes.

## Removing folders

To remove a folder from your library, right-click the folder in the left navigation column and select “Remove”.
![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452742636_512500367954586_2406993045507463854_n.png?_nc_cat=103&ccb=1-7&_nc_sid=e280be&_nc_ohc=HMQXIBNjxCUQ7kNvwFA5RXl&_nc_oc=Adm0ebSO-GiNFHcpF7gTGQxv-sJAYoeG6A94-v_ecRltc871R7WiEIy7Hby9u81Lvxk&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=ry7IOLvf20Hf2Jru2SXokA&oh=00_AfhlLjjU4wkHwkrU8NQyomDpbQxmtJsU3v57bB9FxFoReg&oe=69311E22)

> **Note:** When deleting a folder, any assets inside will be moved into the “Removed Assets” folder.

## Moving assets to different folders

To move an asset into a different folder, right-click the asset card  and select “Move” and then fill out the modal dialog.
