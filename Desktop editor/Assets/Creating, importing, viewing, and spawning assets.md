# Creating, importing, viewing, and spawning assets

A world is only as good as its assets, and in this article you will learn how to create, import, spawn, and view assets for your world.

## Creating assets

![Image shows a close-up of the desktop editor with the create asset menu option highlighted](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/488460142_688083043729650_6946657774903996174_n.png?_nc_cat=104&ccb=1-7&_nc_sid=e280be&_nc_ohc=6A0W1xlsHNUQ7kNvwH_Voo9&_nc_oc=Adl8Dh-2lURB4QxdbTW5KLprBLLS2EgpLl7gZ3QRB7cKAr6bVNJgZWwZAPde-OTlCCw&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=NGwo5ASVeNtPL64nbMF5Cg&oh=00_AfiIeKYMH1AMD811ez5qW9X0-ztOK7Fe4UVON-um53Fhuw&oe=692EA282)

To create an asset:

- Right-click an object in the hierarchy list.
- Fill in the **Create New Asset** fields and click **Create**.

When the asset is created, a notification and a new asset card will appear.

## Importing assets

![Gif shows a user quickly importing an asset via the desktop editor](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/488216084_688083060396315_1099208016929221640_n.png?_nc_cat=101&ccb=1-7&_nc_sid=e280be&_nc_ohc=5UUdBeziyLUQ7kNvwHz1T0Q&_nc_oc=AdlCkP9rllji00B-yQhq-KgZ0DWwbf3862hP0NuA7GKyGiyCnIGqBqRWU91VcMZhJZI&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=NGwo5ASVeNtPL64nbMF5Cg&oh=00_AfgDYJ8r6EooRkE0-JLNsf-4EGyo1pj-wX6NrTwBYvcf0A&oe=692E91F7)

To import an asset:

- Click the **Add New** button and select the desired asset type.
- When the modal dialog appears, select the desired file(s), resolve any errors, and click **Import**.

Note

 When uploading multiple 3D models, ensure that every texture can be used by all the FBX files.

As your imports are being processed, asset cards with a clock icon will appear for each new item.

![Image shows an asset card with a clock icon in the upper right-hand corner](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/487454740_688083057062982_3185833838938401356_n.png?_nc_cat=110&ccb=1-7&_nc_sid=e280be&_nc_ohc=JRVZGnICuRYQ7kNvwF1W-3A&_nc_oc=AdnoSKUMPyk3s4lxsSAWVun72nUHovAttZmZK7xUnoW_2ls5gn8YNU72jCopthJI5bI&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=NGwo5ASVeNtPL64nbMF5Cg&oh=00_AfhDo_8Vcei9qGwxkwuVLtD8yo5p_06Ff1d0JBWO99anFQ&oe=692E911B)

When all your assets have been processed, a notification will appear to confirm whether or not your imports were successful. If your asset had problems while processing, an error or warning icon will appear. To view the asset problems, click the asset card that has an error or warning icon.

![Image shows two asset cards, one with the error icon and the other with the warning icon](https://scontent-dfw5-3.xx.fbcdn.net/v/t39.2365-6/487474566_688083047062983_3792634578100622688_n.png?_nc_cat=109&ccb=1-7&_nc_sid=e280be&_nc_ohc=H5wEN9s1y2UQ7kNvwEDfjjC&_nc_oc=AdkqN8jm8ksv8gbYGrMFuR7xdIZJOPZQNMxN_PbP0Crh-F_uKoMIzzddpdAlbZdnUKM&_nc_zt=14&_nc_ht=scontent-dfw5-3.xx&_nc_gid=NGwo5ASVeNtPL64nbMF5Cg&oh=00_AfhfVgdop5r93zAhn1EXZyXi7onT255jF56fZXKCA57FkA&oe=692E9FD6)

## Viewing asset details

![Gif shows a user clicking on an asset card to view the asset details](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/488628631_688083053729649_726452043044237981_n.png?_nc_cat=102&ccb=1-7&_nc_sid=e280be&_nc_ohc=sJCaaeD9IiMQ7kNvwGEDWjs&_nc_oc=Admqq1tGS6aRpUqbHtJKPr6KqbShyEHOf1Wa3V0FOpNZoFJogAMZ3Mn9gMtUhKryUVY&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=NGwo5ASVeNtPL64nbMF5Cg&oh=00_AfhogpzKu-_CQSgRvSRxXHdgjEVoaKWUHFo2iTrhrnwAEg&oe=692EACF1)

To view asset details:

- Click an asset card.

Asset names and descriptions can be changed from the asset details screen. To edit the asset name or description, select the appropriate field, enter the new value, and click **Enter**.

## Spawning assets

![Gif shows a user spawning an asset using the drag-and-drop method](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/487478362_688083050396316_1036868015224705166_n.png?_nc_cat=104&ccb=1-7&_nc_sid=e280be&_nc_ohc=7ccduuVQ6PAQ7kNvwGW_BH2&_nc_oc=AdlQBYxvHzLiml1w-jc1jiJHPrDLIyew0hXNfwXTsgmi-9bwbVPeDWzEGRkyE2Pfiqk&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=NGwo5ASVeNtPL64nbMF5Cg&oh=00_AfhXgvSdEBQ1k8FvF3KV7NePgYWQr-ghjKR2Jq7BU3wLrQ&oe=692E9D7A)

To spawn an asset:

- Right-click an asset card and select **Place**, or;
- Drag-and-drop the asset card into the viewport.
