# Renaming Objects

This feature enables you to rename objects and gizmos in the hierarchy, so they are easy to find later.

![renamed.gif](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452415597_512527127951910_1338217283483600511_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=LNlGmNROcxAQ7kNvwHqwbbI&_nc_oc=AdlLPbLChlUhC5XDSsZ7lRa5X7q_BQeM67HxjL7dseHo6BhU-wSqG_17ITmdFQue7kw&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=YFqyLUZDPSexewlgrL-V8Q&oh=00_Afgm3G-t01jbaGqHSMk3hJ3fld2TSNEKFw6iwDZtqEK6Nw&oe=692EA138)

Follow these steps to rename objects in the hierarchy:

- Select any entity in the world (*single left click*).
  ![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452869683_512527227951900_7209877553680932515_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=3ifLPJ3P0uIQ7kNvwENrY5A&_nc_oc=AdlmSgLVsC5mrq3awv3iOu2S35_Pgn7S4cOhF19cbOPYyytAD0GzYWEukgS6xjIEy44&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=YFqyLUZDPSexewlgrL-V8Q&oh=00_AfgNPfga-DBGRbImJPm29YlXIw8ozya8t9MgXyNmeoB1bw&oe=692EAB81)
- Click on the selected entity again (*single left click*) or Press F2.
  ![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452578812_512527224618567_8996967759305590928_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=a3GLYjqIqa4Q7kNvwGbwtx1&_nc_oc=AdmleRT478w-AOEfqHW8aYVOXTpX7O2D4cP6slVF--8QaYs4u0yCaxli1bRutnvj9XM&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=YFqyLUZDPSexewlgrL-V8Q&oh=00_AfhX0a0DPalLXYx0wvks1765z2DzLvu9COqN6c-0OrbaFA&oe=692EB3F2)
  You can also right click and select **Rename** from the popup menu.
  ![Rename option from right click menu](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/475456016_643175448220410_4577554162907610986_n.png?_nc_cat=106&ccb=1-7&_nc_sid=e280be&_nc_ohc=tq450HV7pkEQ7kNvwFbkIpw&_nc_oc=AdloniZP870sMo0e1NwsH7HhX1wNYgdYhbbCn8R90TcKoIxYu7KIS8GLPIS2dcmyq5A&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=YFqyLUZDPSexewlgrL-V8Q&oh=00_AfjUGHSUjUK3ZMXTmBUsvReSDvVN1jdFMOzHaoRTFhzO3Q&oe=692E99B3)
- Type in the new name.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452520176_512527107951912_1297209859359763167_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=IeLRPsuI3MIQ7kNvwGH5Pzd&_nc_oc=Adk6l4X8Tzpd0K8O2v6xEyWX0uVShcAH-d327TeUR-ihvJTG0RB72uzXmWDnoB8xsYU&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=YFqyLUZDPSexewlgrL-V8Q&oh=00_Afj4mmTd-zLe8pJDNf8j0liaHAoquc_PkyNlySBwK7C6QQ&oe=692E9810)
- Press Enter or unfocus the text input to save.
  ![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452694614_512527124618577_9191890227449134237_n.png?_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=1X3-DoVICpcQ7kNvwGbGBmV&_nc_oc=Adkm-XlXuUAtljqU5zX6u7bDOQxSvI0g9AHEkzkadxwslezJgvUqu4dvl6bbTzYO4vs&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=YFqyLUZDPSexewlgrL-V8Q&oh=00_Afigfi2Zu4Lc97gudJKWfDcIi_x55BRKiqwofLZRlpYvsQ&oe=692EC238)
