# Renaming Objects

This feature enables you to rename objects and gizmos in the hierarchy, so they are easy to find later.

![renamed.gif](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452415597_512527127951910_1338217283483600511_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=l2Q5sWxo8Y4Q7kNvwE2yywj&_nc_oc=AdlAVAItFmOjCRQj0kMEkYtnKMqRGWtlu2skkQBiQs2uoG3Jj-7SB6vsMnL4mqKWEqI&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=J_LzDrPJrbLVbzrqv3_gMg&oh=00_AfmKs3x8AgZ3fL1d1Dec5ej10L_3ADZvfsPLxk-hnHOjgQ&oe=694BDA78)

Follow these steps to rename objects in the hierarchy:

- Select any entity in the world (*single left click*).
  ![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452869683_512527227951900_7209877553680932515_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=Q18gYnuPi4UQ7kNvwH1RSua&_nc_oc=Admw1ok9TB1SfvitbWfLtV6dhDQr8x8OVj6gbKm9L4vqvMh8jPoDACHLtFJaBe82Q-k&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=J_LzDrPJrbLVbzrqv3_gMg&oh=00_AfmScY5p-hJdTgv0VAtudZ8GEUs1UQ155r5ON7VEXx-P_A&oe=694BE4C1)
- Click on the selected entity again (*single left click*) or Press F2.
  ![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452578812_512527224618567_8996967759305590928_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=RKKWLjJ8vJcQ7kNvwHRAQjA&_nc_oc=Admcp9-tRC2yQ3nqWEb3mWrvi8-kyMyaMnLWUaDXFkAublTRl3nV6bJqqAjMNbUWn9U&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=J_LzDrPJrbLVbzrqv3_gMg&oh=00_Afmt__uEZk7tlrrYWsi2ZfQ9Kr9QwJ4yxRW2ONyXCIlMIg&oe=694BB4F2)
  You can also right click and select **Rename** from the popup menu.
  ![Rename option from right click menu](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/475456016_643175448220410_4577554162907610986_n.png?_nc_cat=106&ccb=1-7&_nc_sid=e280be&_nc_ohc=xdFYmhP357cQ7kNvwEZaOl2&_nc_oc=Adl22OmIEv-hgRXexOmwT_oXTtwNJ8Sg67m6EOO0NCkznNVgeJFoE_N172dP8PFdCSY&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=J_LzDrPJrbLVbzrqv3_gMg&oh=00_AfmTfVR8_31ve5vs91sTLTvxo3_5asnnR49MuUZAo1UaFA&oe=694BD2F3)
- Type in the new name.
  ![](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452520176_512527107951912_1297209859359763167_n.png?_nc_cat=107&ccb=1-7&_nc_sid=e280be&_nc_ohc=HbC_jDOEvHQQ7kNvwElxKwO&_nc_oc=AdlKNocqSL9-JngvI1F3i3zOQdoyDW7r3GfSeQgF4-667HT_3vgVO6sXDVEghz53hkk&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=J_LzDrPJrbLVbzrqv3_gMg&oh=00_AfkMnD67L1U9-YZo2IsaU_SffTPLsNBDEmL831iHdoNnNQ&oe=694BD150)
- Press Enter or unfocus the text input to save.
  ![](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452694614_512527124618577_9191890227449134237_n.png?_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=2IJjaT-bgaAQ7kNvwGTW5cU&_nc_oc=Adno3FzrHmMqM3S7NbgEipPMRXl0Ly5UeZ2XqFHDoHSnchnFZ3IO7y5TC6ACRZai5rU&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=J_LzDrPJrbLVbzrqv3_gMg&oh=00_Aflsnqj9UfQ2FdpBxskabDmRKs3ju18qOx09hwtloVciGg&oe=694BC338)
