# Panels and Tabs in the desktop editor
...
## Hierarchy panel
...
 The Hierarchy panel displays the list of objects in the current scene, such as 3D models and
script components. When you add objects to your scene, entries for them also
appear in your Hierarchy. You can use this panel to sort and group the objects for selecting or
filtering. For more information, see the [Hiearchy Panel Overview](https://developers.meta.com/horizon-worlds/learn/documentation/desktop-editor/hierarchy-window/hierarchy-window-overview).

## Console tab
...
 The Console tab opens a panel that displays a running list of status messages generated by
the editor and the Meta Horizon Worlds runtime. These messages contain
information intended to help you find and fix problems with your world. For more information, see the [Desktop editor Console](https://developers.meta.com/horizon-worlds/learn/documentation/desktop-editor/getting-started/editor-console) page.

• Hierarchy panel
