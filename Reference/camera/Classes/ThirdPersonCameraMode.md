# ThirdPersonCameraMode Class

Extends *[ICameraMode](../Interfaces/ICameraMode.md)*

Manipulates runtime properties of cameras in third person mode, which uses a camera view that follows the local player avatar.

## Signature

```
export declare class ThirdPersonCameraMode implements ICameraMode
```

## Remarks

The [Camera.setCameraModeThirdPerson()](Camera.md#setcameramodethirdperson) method enables third person camera mode. For more information on setting camera modes at runtime, see the [Camera](../../../Mobile%20and%20web/TypeScript%20APIs%20for%20mobile/Camera.md) guide.
