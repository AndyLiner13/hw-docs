# FirstPersonCameraMode Class

Extends *[ICameraMode](/hw-mcp-tools/documentation/hw-docs/Reference/camera/Interfaces/ICameraMode.md)*

Manipulates runtime properties of cameras in first person mode, which uses a camera view from the eyes of the player avatar.

## Signature

```
export declare class FirstPersonCameraMode implements ICameraMode
```

## Remarks

The [Camera.setCameraModeFirstPerson()](/hw-mcp-tools/documentation/hw-docs/Reference/camera/Classes/Camera.md#setcameramodefirstperson) method enables first person camera mode. For more information on setting camera modes at runtime, see the [Camera](/hw-mcp-tools/documentation/hw-docs/Mobile%20and%20web/TypeScript%20APIs%20for%20mobile/Camera.md) guide.
