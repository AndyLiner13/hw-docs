# VideoCaptureNotifications Enum

> Warning: This API is now obsolete.
> This module is deprecated and will be removed in a future version.

The flags for notification types to show players after video recording completes.

## Signature

```
export declare enum VideoCaptureNotifications
```

## Enumeration Members

| Member | Value | Description |
| --- | --- | --- |
| All | `1` | Show all notifications. |
| None | `0` | Don't show any notifications. |
| Toast | `1` | Show a toast notification. |
