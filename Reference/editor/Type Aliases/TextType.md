# TextType type

The possible size and weight properties for a text element.

## Signature

```
export declare type TextType = 'XXLMedium' | 'XLMedium' | 'LMedium' | 'MLBold' | 'MLRegular' | 'MBold' | 'MRegular' | 'SBold' | 'SRegular' | 'XSRegular';
```
