# ViewProps type

Represents the props of a [View()](/hw-mcp-tools/documentation/hw-docs/Reference/ui/Functions/View.md) component.

## Signature

```
export declare type ViewProps = {
    children?: UIChildren;
    style?: ViewStyle;
};
```
