# ImageProps type

Represents the props of an [Image](/hw-docs/Reference/ui/Functions/Image_2.md) component.

## Signature

```
export declare type ImageProps = {
    source?: Bindable<ImageSource | null>;
    style?: ImageStyle;
};
```
