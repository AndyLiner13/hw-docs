# TimingAnimationConfig type

Defines how to [animate](../Abstract%20Classes/Animation_2.md) a value along an easing curve.

## Signature

```
export declare type TimingAnimationConfig = {
    duration?: number;
    easing?: Easing;
};
```

## References

[Easing](../Classes/Easing.md)

## Remarks

Type Parameters:
duration - The duration of the animation.
easing - The function that defines the easing curve.
