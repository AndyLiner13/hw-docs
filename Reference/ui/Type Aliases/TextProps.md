# TextProps type

Represents the props of a [text](/hw-docs/Reference/ui/Functions/Text_2.md) component.

## Signature

```
export declare type TextProps = {
    text: Bindable<string | LocalizableText>;
    numberOfLines?: number;
    style?: TextStyle;
};
```
