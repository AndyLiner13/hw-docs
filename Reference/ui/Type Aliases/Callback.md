# Callback type

Represents a callback function interface for a [Player](/hw-mcp-tools/documentation/hw-docs/Reference/core/Classes/Player.md) object.

## Signature

```
export declare type Callback = (player: Player) => void;
```

## Remarks

Type Parameters:
player - The player associated with the callback.
