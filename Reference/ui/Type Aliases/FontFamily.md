# FontFamily type

Defines the available fonts for a [Text](/hw-mcp-tools/documentation/hw-docs/Reference/ui/Functions/Text_2.md) component in a UI panel.

## Signature

```
export declare type FontFamily = 'Anton' | 'Bangers' | 'Kallisto' | 'Optimistic' | 'Oswald' | 'Roboto' | 'Roboto-Mono';
```

## Remarks

For descriptions of the available styles, see [Custom UI Styles](/hw-mcp-tools/documentation/hw-docs/Desktop%20editor/Custom%20UI/Custom%20UI%20Styles.md#textstyle).
