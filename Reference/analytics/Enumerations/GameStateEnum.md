# GameStateEnum Enum

The shared game states for a world instance. This is not player-specific.

## Signature

```
export declare enum GameStateEnum
```

## Enumeration Members

| Member | Value | Description |
| --- | --- | --- |
| ACTIVE | `"ACTIVE"` | The game is active. |
| BETWEEN\_ROUNDS | `"BETWEEN_ROUNDS"` | The game is in between rounds. |
| BETWEEN\_STAGES | `"BETWEEN_STAGES"` | The game is in between stages. |
| NEVER\_STARTED | `"NEVER_STARTED"` | The game has not started. |
| NONE | `"NONE"` | No game state. |
