# TooltipAnchorLocation Enum

The location where a tooltip is anchored.

## Signature

```
export declare enum TooltipAnchorLocation
```

## Enumeration Members

| Member | Value | Description |
| --- | --- | --- |
| LEFT\_WRIST | `"LEFT_WRIST"` | The tooltip is anchored at the left wrist. |
| RIGHT\_WRIST | `"RIGHT_WRIST"` | The tooltip is anchored at the right wrist. |
| TORSO | `"TORSO"` | The tooltip is anchored at the torso. |
