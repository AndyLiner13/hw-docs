# AnimationCallbackReason type

Represents the [reason](../Enumerations/AnimationCallbackReasons.md) that an [AnimationCallback](AnimationCallback.md) or [AvatarGripPoseAnimationCallback](AvatarGripPoseAnimationCallback.md) function triggered, such as if the animation is starting or stopping.

## Signature

```
export declare type AnimationCallbackReason = AnimationCallbackReasons;
```
