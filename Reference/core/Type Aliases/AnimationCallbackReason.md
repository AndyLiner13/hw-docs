# AnimationCallbackReason type

Represents the [reason](/hw-mcp-tools/documentation/hw-docs/Reference/core/Enumerations/AnimationCallbackReasons.md) that an [AnimationCallback](/hw-mcp-tools/documentation/hw-docs/Reference/core/Type%20Aliases/AnimationCallback.md) or [AvatarGripPoseAnimationCallback](/hw-mcp-tools/documentation/hw-docs/Reference/core/Type%20Aliases/AvatarGripPoseAnimationCallback.md) function triggered, such as if the animation is starting or stopping.

## Signature

```
export declare type AnimationCallbackReason = AnimationCallbackReasons;
```
