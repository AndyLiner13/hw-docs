# LocomotionOptions type

The options used when a movement command is issued to the NPC.

## Signature

```
export declare type LocomotionOptions = {
    movementSpeed?: number;
    travelTime?: number;
    acceleration?: number;
    deceleration?: number;
    faceMovementDirection?: boolean;
};
```
