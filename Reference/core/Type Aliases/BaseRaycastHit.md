# BaseRaycastHit type

The base class for the result of a [raycast](/hw-docs/Reference/core/Classes/RaycastGizmo.md#raycast) collision.

## Signature

```
export declare type BaseRaycastHit = {
    distance: number;
    hitPoint: Vec3;
    normal: Vec3;
};
```
