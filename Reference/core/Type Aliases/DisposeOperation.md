# DisposeOperation type

A callback used to perform a single registered dispose operation, either automatically at the dispose time of the [DisposableObject](/hw-mcp-tools/documentation/hw-docs/Reference/core/Interfaces/DisposableObject.md) instance, or manually before the dispose operation.

## Signature

```
export declare type DisposeOperation = () => void;
```
