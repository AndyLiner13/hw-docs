# StopAnimationOptions type

The options for the [Player.stopAvatarAnimation()](../Classes/Player.md#stopavataranimation) method, which stops a non-looping animation on an avatar.

## Signature

```
export declare type StopAnimationOptions = {
    fadeOutDuration?: number;
};
```
