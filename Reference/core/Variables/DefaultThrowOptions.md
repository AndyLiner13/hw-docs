# DefaultThrowOptions Variable

The default values for the [ThrowOptions](/hw-mcp-tools/documentation/hw-docs/Reference/core/Type%20Aliases/ThrowOptions.md) type, which defines the throwing behaviour when using the [Player.throwHeldItem()](/hw-mcp-tools/documentation/hw-docs/Reference/core/Classes/Player.md#throwhelditem) API.

## Signature

```
DefaultThrowOptions: ThrowOptions
```

speed: 20
