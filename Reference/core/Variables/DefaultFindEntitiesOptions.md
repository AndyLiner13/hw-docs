# DefaultFindEntitiesOptions Variable

Default ptions for the [World.findEntities()](/hw-docs/Reference/core/Classes/World.md) method.

## Signature

```
DefaultFindEntitiesOptions: FindEntitiesOptions
```

## Remarks

rootEntity - defaults to undefined, meaning the entire hierarchy will be searched matchOperation - defaults to [EntityNameMatchOperation.Exact](/hw-docs/Reference/core/Enumerations/EntityNameMatchOperation.md)
