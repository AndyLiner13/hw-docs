# DefaultFocusedInteractionTapOptions Variable

The default values for the [FocusedInteractionTapOptions](/hw-docs/Reference/core/Type%20Aliases/FocusedInteractionTapOptions.md) type, which defines the [visual](/hw-docs/Reference/core/Classes/FocusedInteraction.md) options when players use tap input.

## Signature

