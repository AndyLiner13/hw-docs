# LEADEBOARD\_SCORE\_MAX\_VALUE Variable

The maximum value allowed for a score on an [ILeaderboards](https://developers.meta.com/horizon-worlds/reference/2.0.0/core_ileaderboards) object.

## Signature

```
LEADEBOARD_SCORE_MAX_VALUE: number
```

## Remarks

This variable is used by the [ILeaderboards.setScoreForPlayer()](https://developers.meta.com/horizon-worlds/reference/2.0.0/core_ileaderboards#setscoreforplayer) method.
