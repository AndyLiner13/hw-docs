# LEADEBOARD\_SCORE\_MAX\_VALUE Variable

The maximum value allowed for a score on an [ILeaderboards](../Interfaces/ILeaderboards.md) object.

## Signature

```
LEADEBOARD_SCORE_MAX_VALUE: number
```

## Remarks

This variable is used by the [ILeaderboards.setScoreForPlayer()](../Interfaces/ILeaderboards.md#setscoreforplayer) method.
