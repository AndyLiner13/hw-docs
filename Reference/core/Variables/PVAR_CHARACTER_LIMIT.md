# PVAR\_CHARACTER\_LIMIT Variable

The character limit for variables when using an [IPersistentStorage](/hw-docs/Reference/core/Interfaces/IPersistentStorage.md) object.

## Signature

```
PVAR_CHARACTER_LIMIT = 10000
```
