# DefaultMigrationOptions Variable

The default values for the [MigrationOptions](/hw-mcp-tools/documentation/hw-docs/Reference/migration/Type%20Aliases/MigrationOptions.md) type, used when no options are provided to the [PlayerMigration.migrateToHorizonApp()](/hw-mcp-tools/documentation/hw-docs/Reference/migration/Classes/PlayerMigration.md#migratetohorizonapp) method.

## Signature

DefaultMigrationOptions: MigrationOptions
