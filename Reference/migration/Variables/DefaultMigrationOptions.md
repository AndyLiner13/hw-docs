# DefaultMigrationOptions Variable

The default values for the [MigrationOptions](https://developers.meta.com/horizon-worlds/reference/2.0.0/migration_migrationoptions) type, used when no options are provided to the [PlayerMigration.migrateToHorizonApp()](https://developers.meta.com/horizon-worlds/reference/2.0.0/migration_playermigration#migratetohorizonapp) method.
