# MobileAppPlatform Enum

The type of mobile app the player is using to play Horizon.

## Signature

```
export declare enum MobileAppPlatform
```

## Enumeration Members

| Member | Value | Description |
| --- | --- | --- |
| META\_HORIZON | `"META_HORIZON"` | The player is playing through the dedicated Meta Horizon app |
| META\_OTHER | `"META_OTHER"` | The player is playing through other Meta Apps like Facebook or Instagram. |
| NOT\_MOBILE | `"NOT_MOBILE"` | The player is not on a mobile device. |
