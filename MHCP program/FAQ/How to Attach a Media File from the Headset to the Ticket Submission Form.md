# How to Attach a Media File from the Headset to the Ticket Submission Form?

**1. In Headset:**

- Open **Apps** from the universal menu.
  ![Apps.PNG](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452954747_512510607953562_5832863781158349036_n.png?_nc_cat=102&ccb=1-7&_nc_sid=e280be&_nc_ohc=8dW7kr7_SA8Q7kNvwG3kzZ6&_nc_oc=Adlnlf6wZjNef4qLNadw8MR7UU4JA74XsjneyFjeD6KdDfQ_vx5mNFR5Y6zUAshRz5o&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=gFyHqIylUuNwP4vixuK22Q&oh=00_AfmYgxk2T4jvH6nMh5a_fz3z6FBUYuP-NMIOUbOtBnbQ7Q&oe=694BEB77)
- Select the **Files** app.
- Navigate to a specific file and select **Sync to Mobile App**.
  ![SynctoMobile.PNG](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452653014_512510604620229_8394962535313930383_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=rCHn_-Zu0ZcQ7kNvwFLArHy&_nc_oc=AdkLZooGVtgLWl-M0hrtPMFrDlzzsN32zGd3-uMdNc2H22jH3RV6YrdXsNmd8-qUCgE&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=gFyHqIylUuNwP4vixuK22Q&oh=00_Afn01gPkYGQXt7JzXTx40YM_C1w0w-YRGdzVu8qAiqs21A&oe=694BCDF8)

**2. From Mobile App**

- Navigate to the **Menu** tab.
- Select the **Devices** button and pair with the headset the media file is saved on.
- Select the media file(s) you want to attach to the submission form.
- Click on the **Share** button on the top right corner
  ![ShareButton.PNG](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452682677_512510601286896_8338315398329531307_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=a1_XwDL1Bj0Q7kNvwH3OgU9&_nc_oc=AdnBHB058NJz9suupIMtzwoLQwFCw7APpuSdiU_zNHBdtwr3GfxIeXzm6FuDHM-m5Pg&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=gFyHqIylUuNwP4vixuK22Q&oh=00_AfnUmShpDAeQ8MetlrxzKoGooBipzFu3YuKjy7jcxu6ijQ&oe=694BC58F)
- Select the **Email** option and email the file to yourself.
- Attach the file from your email to ticket submission.
