# How to Attach a Media File from the Headset to the Ticket Submission Form?

**1. In Headset:**

- Open **Apps** from the universal menu.
  ![Apps.PNG](https://scontent-dfw5-2.xx.fbcdn.net/v/t39.2365-6/452954747_512510607953562_5832863781158349036_n.png?_nc_cat=102&ccb=1-7&_nc_sid=e280be&_nc_ohc=L6fUqF5dV38Q7kNvwHa7J4h&_nc_oc=Adn3Pe6d72_aq7_QOI-SWWHeSnNLCE_rjmXlaDVuo47o0DwAIb5JzcmKD1JGLy3ugZ4TQ6daR8O7QM8fltkKMv-5&_nc_zt=14&_nc_ht=scontent-dfw5-2.xx&_nc_gid=UHqF-z2tIfauxjbtdtTHRg&oh=00_AflzEMZKgacR-V2JcJ2YYhsDxcBnfN5hHfyj3t16sPb3WA&oe=69559677)
- Select the **Files** app.
- Navigate to a specific file and select **Sync to Mobile App**.
  ![SynctoMobile.PNG](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452653014_512510604620229_8394962535313930383_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=QntTw4oBl8cQ7kNvwGQELdk&_nc_oc=AdmaCokVD91U_i2a62v2r60XR9UVY5NuadB_lu_rCQLO_ufM2AKlSB5k8P0ztN0buukUQY_mAyQT8IYyH6UbunQB&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=UHqF-z2tIfauxjbtdtTHRg&oh=00_Afn9OlnTZIcbr4MQkYMVbtK8_IdN9YUjsMDGZDsFO6_NRA&oe=695578F8)

**2. From Mobile App**

- Navigate to the **Menu** tab.
- Select the **Devices** button and pair with the headset the media file is saved on.
- Select the media file(s) you want to attach to the submission form.
- Click on the **Share** button on the top right corner
  ![ShareButton.PNG](https://scontent-dfw5-1.xx.fbcdn.net/v/t39.2365-6/452682677_512510601286896_8338315398329531307_n.png?_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=a1KY2zXIjw0Q7kNvwFeb4_a&_nc_oc=Adm6A9yiouDgfZZdmnlYqZ51YkArNBspMyiqP39UxyFmZWzwFJVHH1uYosDsj3Fx1a1xZeUPW_q30AfGJRztezVM&_nc_zt=14&_nc_ht=scontent-dfw5-1.xx&_nc_gid=UHqF-z2tIfauxjbtdtTHRg&oh=00_AfmdACwTtdB10y6fASIlmIVBlgu-0FDfTIgkJvM0Vss_jA&oe=6955708F)
- Select the **Email** option and email the file to yourself.
- Attach the file from your email to ticket submission.
